# laravel_app_01
This is first CRUD application using,
- PHP Laravel framework
- mySQL
- Bootstrap 4
- jQuery

To setup the project,
1. Clone the git repository.
2. Run composer install commad.
3. Run npm install command.
4. Create a database 'laravel_mysql_app_01'.
5. Run migrations.
6. Serve the app.
